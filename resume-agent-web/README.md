# Resume Agent — Web Client

צד לקוח (Frontend) עבור פרויקט Resume Agent, בנוי ב-**React + TypeScript + Vite**.
נפרד לחלוטין מהבק-אנד ומנוהל כריפוזיטורי עצמאי.

## תכונות

- **העלאת קורות חיים** — גרירה ושחרור (Drag & Drop) או בחירת קובץ (PDF / DOC / DOCX, עד 10MB)
- **זיכרון קבצים קודמים** — כל הקבצים נשמרים מקומית בדפדפן (IndexedDB) וזמינים גם אחרי סגירה
- **קורות חיים פעילים** — הקובץ הפעיל נשלח אוטומטית לסוכן ומשמש בחיפוש העבודה
- **היסטוריה מלאה** — צפייה, הורדה ומחיקה של כל גרסה קודמת
- **טאב "משרות"** — הרצת אלגוריתם חיפוש העבודה מהאתר, מעקב התקדמות חי (כולל יומן),
  וטבלת המשרות עם ציוני ההתאמה, הסברי ה-AI וקישור לכל משרה
- ממשק בעברית מלא (RTL)

## הרצה

1. הפעל את שרת ה-API של הסוכן (בריפו `ai-job-agent`):

```bash
python src/api_server.py     # http://localhost:8000
```

2. הפעל את האתר:

```bash
npm install
npm run dev
```

האתר יעלה בכתובת http://localhost:5173. כתובת אחרת ל-API אפשר להגדיר
עם משתנה הסביבה `VITE_API_URL`.

האתר עובד גם בלי השרת — העלאת קורות חיים והיסטוריה נשמרות מקומית,
וטאב המשרות יציג הודעה שהסוכן לא זמין.

## בנייה לפרודקשן

```bash
npm run build
npm run preview
```

## מבנה הפרויקט

```
src/
  App.tsx                  # מסך ראשי + טאבים + חיווי חיבור לשרת
  components/
    UploadCard.tsx         # אזור העלאה (drag & drop)
    CvHistory.tsx          # רשימת קורות החיים השמורים
    JobsPage.tsx           # הרצת הפייפליין + טבלת המשרות
  lib/
    cvStorage.ts           # שמירה מקומית ב-IndexedDB
    api.ts                 # קריאות ל-API של הסוכן (FastAPI)
  index.css                # עיצוב
```
